
##HTTP 优化

* gzip 压缩
    因为gzip对于含有重复'单词'的文本文件,压缩率非常高
* 两个细节 
   ```
   一个css 资源请求的体积是36.4KB(这是gzip压缩后的),解压缩后css实际内容有264KB,压缩率达13.8%
   整个连接的建立花费30%的时间,发出请求到等待收到第一个字节回复花费20%的时间,下载css资源的内容花费了50%的时间
   ```
